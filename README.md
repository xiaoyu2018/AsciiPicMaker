# **AsciiPicMaker**
*一个字符画转换工具*


## 功能
1. 视频处理
    + 视频抽帧
    + 帧合成视频
2. 字符画转换
    + 彩色字符画转换
    + 黑白字符画转换
    + 实时字符画视频生成
    + 离线字符画视频生成
